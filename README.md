# snow-scene-opengl

三维图形程序设计课程实验：雪中即景

## 实验内容
- 构建雪景主题场景
- 包含房屋、雪人、树木、围栏、路灯、树桩等对象
- 支持白天/黑夜切换
- 支持下雪动画与树枝摆动
- 支持树桩交互操作

## 交互说明
- W / S：视角上移 / 下移
- A / D：视角左移 / 右移
- N：白天 / 黑夜切换
- F：下雪 / 停雪切换
- 鼠标左键拖动：平移树桩
- Ctrl：旋转树桩
- 方向键：拉伸树桩
- 鼠标右键 / R：恢复树桩初始状态

## 开发环境
- OpenGL 3.3 Core
- GLFW
- GLAD
- GLM
- Visual Studio
